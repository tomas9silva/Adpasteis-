## Como publicar

No GitHub, envia estes ficheiros e pastas mantendo a estrutura:

- pasta `css`
- pasta `js`
- `index.html`
- `README.md`

Não coloques todos os ficheiros soltos se o `index.html` estiver a chamar `css/style.css` e `js/app.js`.
