# StatS — AD Pastéis (modular real)

Esta versão divide o código em ficheiros por área.

## Estrutura

- `index.html` — estrutura da app
- `css/style.css` — design
- `js/app.js` — núcleo, variáveis e arranque
- `js/dashboardIndividual.js` — Dashboard Individual
- `js/dashboardColetivo.js` — Dashboard Coletivo
- `js/heatmap.js` — Heat Map
- `js/minutos.js` — Contagem de minutos
- `js/jogo.js` — Jogo, marcador e cronómetro
- `js/arena.js` — Campo/Arena
- `js/treino.js` — Treino
- `js/storage.js` — Guardar/exportar dados
- `js/ui.js` — Menus e modais
- `js/utils.js` — Funções auxiliares

## Como atualizar no GitHub

Substitui a pasta `js`, a pasta `css` e o `index.html` pelos ficheiros deste pacote.
